# a1z26-cipher-tool
This project implements a basic encryption and decryption tool based on the A1Z26 algorithm, where letters are mapped to numbers (A = 1, B = 2, ..., Z = 26) and shifted by a specified amount.

   cd a1z26-cipher-tool
